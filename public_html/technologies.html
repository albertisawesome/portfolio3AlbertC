The Technology that we use in this project is Bootstrap, Html and css. Bootstrap is a program that makes html look nice and is easy to do complicated things that would have took forever with netbeans. You can make carousels,jombotrons, and dropdowns. Html stands for Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. CSS stands for Cascading Style Sheets, (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language. a technique of loading a program into a computer by means of a few initial instructions that enable the introduction of the rest of the program from an input device.
